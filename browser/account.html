So you want Tweet counts huh?

In order to count the number of times a URL has been mentioned on Twitter we need to use Twitter's Search API. Access to the API is restricted so to make sure you have consistantly up-to-date counts we need you to create a new Twitter application and provide us with the access tokens. This will allow OpenShare to query the Twitter API on your behalf and count those URLs.

Step 1: Create a new Twitter application, give it a name, description (e.g 'OpenShare Counter') and your company website or https://openshare.social. You'll need to add your phone number to your Twitter account before you can create an application.

Create Twitter Application

Step 2: Click the "Keys and Access Tokens" tab and paste your consumer keys here. If you click the "permissions" tabs you can set the access level of the application to "Read only" as we do not need to write to the API on your behalf. what is this?

Step 3: Paste the URLs you want to count here
